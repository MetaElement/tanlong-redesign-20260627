# 坦隆环境官网整套页面效果

生成日期：2026-06-27

## 页面范围

1. 首页 / Home：首屏承接 Poten 式沉浸感，但信息改为坦隆的钢渣处理、余热回收、专利与项目证明。
2. 关于我们 / About：把品牌理念、技术领军、信用资质和落地规模前置，建立 B2B 官网的第一层可信度。
3. 产品方案 / Solutions：用三张核心业务图形成方案入口：钢渣处理、余热回收、低碳数据看板，并补足工程服务流程。
4. 核心技术 / Technology：技术页用流程图和专利数据表达硬核能力，避免只放抽象口号。
5. 案例中心 / Cases：案例页不伪造客户 Logo，用应用场景、工况和落地数量建立可信但可发布的证明层。
6. 资源与联系 / Resources & Contact：把资料下载、专家咨询、项目周期和线索表单整合成转化页，服务官网商业目标。
7. 钢渣方案详情 / Slag Treatment：方案详情页把客户痛点、工艺亮点、适用场景和可量化证明集中到一屏。
8. 余热回收详情 / Heat Recovery：余热详情页突出换热系统、能源回收、运行监控和产线安全边界。
9. 数字化看板详情 / Digital Dashboard：数字化详情页用仪表盘化表达能效、产线状态、运维和低碳指标，不伪造真实数据。
10. 案例详情 / Case Detail：案例详情页用“背景、方案、结果、可复用经验”组织内容，不写未授权客户名。
11. 资源文章 / Resource Article：资源文章页服务 SEO、销售赋能和技术决策，不只是新闻列表。
12. 服务流程 / Service Process：服务流程页把咨询、调研、方案、商务、交付、运维复盘形成官网转化闭环。

## 设计规则

- 统一沿用坦隆环境新 LOGO、顶部导航、AAA 信用标识、深水蓝与清透青绿配色。
- 背景与业务图片仅作为视觉层，所有中文、英文、指标、CTA 均由 HTML/CSS 确定性渲染。
- 案例页不伪造客户名称或客户 Logo，使用“应用场景 + 工况 + 落地数量”的安全表达。
- 输出规格统一为 1920x1080 PNG，适合做官网视觉评审与方案汇报。
